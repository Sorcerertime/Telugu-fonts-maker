HOW TO RUN THIS PROJECT
=======================

OPTION 1: Open directly in browser
-----------------------------------
1. Double-click on "index.html" 
2. It will open in your default web browser
3. Start using the Telugu text editor immediately!

OPTION 2: Use a local web server (recommended)
-----------------------------------------------
If you have Python installed:
  python -m http.server 8000

Then open: http://localhost:8000

If you have Node.js installed:
  npx serve

Then open the URL shown in terminal

OPTION 3: Use VS Code Live Server
----------------------------------
1. Install "Live Server" extension in VS Code
2. Right-click on index.html
3. Select "Open with Live Server"

HOW TO USE
==========
1. Type Telugu text in the input box
2. Choose a Telugu font from dropdown
3. Adjust size, color, spacing
4. Upload a background image (optional)
5. Drag text to position it
6. Click "Download PNG" to save

FEATURES
========
✓ 6 Google Telugu fonts + custom font upload
✓ Text color, size, spacing controls
✓ Letter spacing (works properly with Telugu)
✓ Line spacing for multi-line text
✓ Text shadow and outline effects
✓ Background image upload or solid color
✓ Drag & drop text positioning
✓ Export in 1080p or 4K resolution
✓ 100% client-side (no server needed)
✓ Works on desktop, tablet, and mobile
